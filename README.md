### 「注文計算アプリ」とは？

品物の売買における

- お品書きの作成
- 合計金額の計算
- 注文の記録

をやってくれるアプリで、企業が使っている様な本格的なシステムは要らないけどちょっとした物が欲しいという人を利用者として想定しています。例えば自営業の飲食店、学園祭のお店、直売をしている農家などです。
